API Documentation

Integrate Dev Card Showcase data into your applications

Getting Started

The Dev Card Showcase API provides programmatic access to contributor data, project information, and community statistics. All API endpoints return JSON responses.

Quick Start

Base URL: https://api.dev-card-showcase.com/v1

All requests must include the Content-Type: application/json header.

Example Request 
curl -X GET "https://api.dev-card-showcase.com/v1/contributors" \
                        -H "Authorization: Bearer YOUR_API_KEY" \
                        -H "Content-Type: application/json"

Authentication

API requests are authenticated using API keys. Include your API key in the Authorization header.

Authentication Header 
Authorization: Bearer YOUR_API_KEY

Getting an API Key

To obtain an API key:

  1. Visit your Developer Dashboard
  2. Click "Generate New API Key"
  3. Copy and securely store your key

Contributors API

Access contributor profiles, statistics, and contribution data.

GET /contributors

Retrieve a list of all contributors with their profile information.

Query Parameters

Parameter Type Required Description
page integer Optional Page number (default: 1)
limit integer Optional Items per page (default: 20, max: 100)
role string Optional Filter by role (developer, contributor, maintainer)
200 OK Response 
{
  "data": [
    {
      "id": "123",
      "name": "John Doe",
      "role": "Frontend Developer",
      "avatar": "https://github.com/johndoe.png",
      "github": "https://github.com/johndoe",
      "bio": "Passionate developer contributing to open source",
      "contributions": 25,
      "joined_date": "2025-01-15T10:30:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 150,
    "pages": 8
  }
}
GET /contributors/{id}

Retrieve detailed information about a specific contributor.

Path Parameters

Parameter Type Required Description
id string Required Contributor ID or GitHub username

Projects API

Access project data, categories, and contributor projects.

GET /projects

Retrieve a list of all projects in the showcase.

Query Parameters

Parameter Type Required Description
category string Optional Filter by project category
technology string Optional Filter by technology stack

Statistics API

Access community statistics and analytics data.

GET /statistics

Get overall community statistics.

200 OK Response 
{
  "total_contributors": 150,
  "total_projects": 500,
  "total_contributions": 1250,
  "active_contributors": 45,
  "top_technologies": ["JavaScript", "Python", "React"],
  "growth_rate": "15%"
}

Rate Limits

API requests are rate limited to ensure fair usage:

  • Free Tier: 1,000 requests per hour
  • Pro Tier: 10,000 requests per hour
  • Enterprise: Custom limits

Rate limit headers are included in all responses:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200

Error Handling

The API uses conventional HTTP response codes to indicate success or failure.

Status Code Description
200 OK - Request successful
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
404 Not Found - Resource doesn't exist
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error
Error Response Format 
{
  "error": {
    "code": "INVALID_API_KEY",
    "message": "The provided API key is invalid",
    "details": "Please check your API key and try again"
  }
}
