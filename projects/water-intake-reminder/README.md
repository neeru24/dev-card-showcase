# Water Intake Reminder

An app to remind users to drink water throughout the day and track daily hydration goals.

## Features
- Set a daily water intake goal (ml)
- Log water intake throughout the day
- Visual progress bar for daily goal
- Reminders to drink water until goal is reached
- View todayâ€™s intake log
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Set your daily water goal.
3. Log water intake as you drink.
4. See your progress and reminders.

## Tech Stack
- HTML, CSS, JavaScript
