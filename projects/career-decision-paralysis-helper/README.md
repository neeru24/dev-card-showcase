# Career Decision Paralysis Helper

A modern, interactive web app to help students and early professionals overcome career indecision. Take a personalized quiz, see your indecision fatigue, and get actionable career recommendations.

## Features
- Multi-step quiz to clarify strengths, interests, and goals
- Visual feedback on indecision/fatigue
- Actionable, personalized recommendations
- Clean, responsive UI

## Usage
Open `index.html` in your browser. Answer the quiz questions and get your career suggestion!

## Project Structure
- `index.html` — Main app page
- `style.css` — App styling
- `app.js` — Quiz logic and recommendations
- `README.md` — Project info and instructions
