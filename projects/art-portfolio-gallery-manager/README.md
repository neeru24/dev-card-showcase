# Art Portfolio & Gallery Manager

A web app for artists to upload, categorize, and showcase their artwork. Features gallery views, tags, exhibition history, sales tracking, and export to PDF/CSV.

## Features
- Upload and categorize artwork
- Gallery views
- Tags
- Exhibition history
- Sales tracking
- Export to PDF/CSV
- Local storage for all data

## Usage
Open index.html in your browser. Add artwork, view your gallery, and manage your portfolio.
