# Mind Map Creator

An interactive tool for building mind maps. Users can add nodes, connect ideas, drag to rearrange, and style branches. Includes export to PNG/SVG/JSON and collaborative editing mode (local).

## Features
- Add, connect, and drag nodes
- Style branches
- Export to PNG, SVG, JSON
- Collaborative editing (local)

## Usage
Open index.html in your browser. Build your mind map, export, and collaborate locally.
