# Developer Unit Converter

An advanced unit converter built for developers.

## Features
- Multiple categories (Data, Time, Speed)
- Binary vs Decimal conversion
- Live conversion
- Swap units
- Copy results
- Conversion history
- Dark/Light theme

## Tech Stack
- HTML
- CSS
- JavaScript

## Usage
1. Select category
2. Choose units
3. Enter value
4. Click Convert
