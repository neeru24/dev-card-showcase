# Browser Storage Comparison Demo

A hands-on demo to understand how different browser storage mechanisms behave.

## Storage Types Covered
- In-Memory (JavaScript variable)
- localStorage
- sessionStorage
- Cookies

## Features
- Store key-value pairs
- Compare persistence behavior
- Clear all storage
- View stored values live
- Action history log

## Learning Outcomes
- Understand storage lifetime
- See persistence differences
- Learn when to use each storage type

## Tech Stack
- HTML
- CSS
- JavaScript

## Usage
1. Enter key and value
2. Select storage type
3. Save
4. Refresh / close tab to observe behavior
