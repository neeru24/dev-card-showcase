# Contributor Dashboard (My Profile)

Personalized dashboard for contributors:
- Stats overview (PRs, commits, streak)
- Contribution activity heatmap
- Earned badges showcase
- Progress bars for next achievements
- Recent contributions timeline
- Active branches/WIP projects
- Quick actions (Create PR, View Issues, Fork)
- Leaderboard position indicator
- Upcoming milestones tracker
- Contribution goals setter
- Export stats as card/badge image
- Theme customization
- Weekly/monthly summary

## Usage
1. Open `index.html` in your browser.
2. View and interact with your contributor stats and goals.
