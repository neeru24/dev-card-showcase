# Personal Finance Dashboard

A tool for tracking expenses, setting budgets, and visualizing spending habits with interactive graphs and category breakdowns.

## Features
- Add expenses with date, category, amount, and notes
- Set and track a monthly budget
- Visualize spending by category (doughnut chart)
- View spending trends over time (line chart)
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Add your expenses and set your monthly budget.
3. View your spending breakdown and trends in the dashboard.

## Tech Stack
- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) for chart visualization
