# Mood-to-Music Room

An immersive web app for mood-based ambient environments with dynamic visuals and music playlists. Select or detect your mood, and the room adapts with visuals and music from Spotify/YouTube.

## Features
- Mood selection (manual/AI detect)
- Dynamic ambient visuals
- Spotify/YouTube playlist integration
- Preset rooms (Focus, Relax, Sleep)

## Usage
Open `index.html` in your browser. Select a mood or use presets. More features coming soon!
