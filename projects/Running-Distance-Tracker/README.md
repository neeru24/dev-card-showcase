# Running Distance Tracker ğŸ‘Ÿ

A beautiful and intuitive running distance and pace tracking UI designed to help runners analyze their performance and improve over time.

## Features

### ğŸ“ Distance & Pace Logging
- Log running distance and time
- Automatic pace calculation (minutes per km)
- Date tracking for historical data

### ğŸ“Š Performance Analytics
- Total distance accumulated
- Average and best pace tracking
- Visual pace chart for recent runs
- Run count statistics

### ğŸ’¡ Running Tips
- Random motivational and performance tips
- Advice on training, form, and injury prevention

### ğŸ“œ Run History
- Recent runs display
- Easy-to-read format with date, distance, time, and pace

## How to Use

1. **Log a Run**: Enter distance (km), time (minutes), and date, then click "Log Run"
2. **View Stats**: Check your total distance, average pace, and best performance
3. **Analyze Progress**: Review the pace chart and run history
4. **Get Tips**: Click "New Tip" for running advice

## Technical Details

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser localStorage for data persistence
- **Charts**: Custom Canvas-based visualization
- **Responsive**: Optimized for desktop and mobile

## Data Privacy

All running data is stored locally in your browser. No information is shared or transmitted.
