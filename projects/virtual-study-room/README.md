# Virtual Study Room

Create online study rooms where users can join, set timers, and work together silently. Includes ambient sounds, chat, and productivity stats to simulate a real study environment.

## Features
- Create or join study rooms
- Pomodoro/custom timer
- Ambient sounds (rain, cafe, forest)
- Chat with room members
- Productivity stats (sessions, minutes)
- All data stored in browser (no backend)

## Usage
1. Enter your name and a room name.
2. Create or join a room.
3. Set a timer and start studying.
4. Use ambient sounds and chat as needed.
5. Track your productivity stats.

## Tech
- HTML, CSS, JavaScript
- All data stored in browser (no backend)
