# Digital Garden for Notes & Ideas

Organize notes and ideas visually, linking related concepts in a digital garden format.

## Features
- Add notes with title and content
- Link notes to show relationships
- Visualize notes and links as a simple graph
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Add notes and ideas.
3. Link related notes to visualize connections.
4. See your digital garden graph update live.

## Tech Stack
- HTML, CSS, JavaScript
