# Fitness Workout Logger

Record workouts, track progress, and visualize improvements over time.

## Features
- Log workouts with date, type, duration, and notes
- Visualize workout duration trends over time (line chart)
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Add your workouts as you complete them.
3. View your workout history and progress chart.

## Tech Stack
- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) for progress chart
