      Help & Documentation

      Getting Started

      The Semantic Relationship Mapping Console allows you to create and visualize semantic relationships between concepts.

      How to Use

      1. Click "Add Node" to create new concepts
      2. Select two nodes and click "Add Relation" to connect them
      3. Use the graph view to visualize relationships
      4. Switch to list view for detailed information
      5. Check analytics for graph metrics

      Relation Types

      • is-a: Hierarchical relationship (e.g., cat is-a animal)
      • part-of: Meronymic relationship (e.g., wheel part-of car)
      • related-to: General association
      • causes: Causal relationship
      • synonym: Equivalent meaning

      Keyboard Shortcuts

      • Ctrl+N: Add new node
      • Ctrl+R: Add relation
      • Ctrl+Z: Undo
      • Delete: Remove selected

      Tips

      Use the zoom controls to navigate large graphs. Right-click on nodes for context menus.
