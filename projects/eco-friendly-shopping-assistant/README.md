# Eco-Friendly Shopping Assistant

A tool that suggests sustainable products, tracks eco-friendly purchases, and educates users on green shopping habits.

## Features
- Search and suggest sustainable products
- Track eco-friendly purchases
- Learn green shopping habits
- View profile info

## Getting Started
Open `index.html` in your browser to use the app.

## Project Structure
- `index.html` — Main HTML file
- `style.css` — Stylesheet
- `app.js` — JavaScript logic
- `README.md` — Project info
