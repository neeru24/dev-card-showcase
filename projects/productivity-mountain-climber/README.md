# Productivity Mountain Climber

Set daily productivity goals and check in to move your climber up the mountain. Consistent productivity unlocks new scenery and rewards. Missed days cause the climber to rest or slide down.

## Features
- Set and track daily productivity goals
- Each check-in moves the climber up
- Consistency unlocks new scenery and rewards
- Missed days cause the climber to rest or slide down
- Modern, beautiful UI

## Usage
1. Open `index.html` in your browser.
2. Set your productivity goal for the day.
3. Check in daily to climb higher!
