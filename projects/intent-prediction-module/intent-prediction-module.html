About Intent Prediction Module

This module uses machine learning to predict user intents from text inputs. It supports single and batch predictions, maintains history, and provides detailed statistics.

Supported Intents:

  • Greeting: Hello, hi, good morning, etc.
  • Question: What, how, why, when, where, who questions.
  • Command: Please do this, create, run, start, etc.
  • Statement: General statements and declarations.
  • Request: Can you, would you, etc.
  • Confirmation: Yes, no, okay, etc.

Technology Stack:

  • TensorFlow.js for machine learning
  • tfjs-vis for visualizations
  • Vanilla JavaScript for logic
  • CSS for styling and animations
