How to Play

  1. Add players using the "Add Player" button or by clicking the + button next to existing players
  2. Players will appear around the bottle in a circle
  3. Click "SPIN THE BOTTLE!" to spin the bottle
  4. The bottle will point to a player when it stops spinning
  5. Use the "Random Dare" button to generate a fun dare for the selected players
  6. Have fun and keep the dares friendly and appropriate!

Remember: This is a game for fun. Always respect others' boundaries!
