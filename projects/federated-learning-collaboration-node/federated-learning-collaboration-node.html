Simulate a federated learning system where multiple client nodes collaboratively train a machine learning model without sharing their private data.

How It Works

In federated learning, each client trains a local model on their data, then shares only the model updates (not the data) with a central server. The server aggregates these updates to improve the global model, which is then distributed back to clients.
