# Gratitude Sky Lanterns

Write daily gratitude notes. Each note releases a lantern into a night sky. The more consistent the gratitude practice, the brighter and more colorful the sky becomes.

## Features
- Write and log daily gratitude notes
- Each note releases a lantern into the sky
- Consistency makes the sky brighter and more colorful
- Gratitude history list
- Modern, beautiful UI

## Usage
1. Open `index.html` in your browser.
2. Write a gratitude note each day.
3. Watch your sky fill with lanterns!
