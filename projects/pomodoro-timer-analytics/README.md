# Pomodoro Timer with Analytics

A productivity timer that logs sessions and provides analytics on focus patterns.

## Features
- Pomodoro timer (25 min focus, 5 min break)
- Start, pause, and reset controls
- Logs each session with date and time
- Analytics: focus sessions per day (last 7 days)
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Start the timer to begin a focus session.
3. View your session log and analytics below the timer.

## Tech Stack
- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) for analytics chart
