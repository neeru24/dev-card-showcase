# Personal Knowledge Base & Wiki

A web app for managing notes, research, and references with hierarchical pages, search, backlinks, markdown support, and export features.

## Features
- Hierarchical pages (tree structure)
- Search across pages and content
- Backlinks (show references to a page)
- Markdown support (basic formatting)
- Export to JSON and Markdown
- Local storage for persistence

## Usage
Open index.html in your browser. Add pages, edit content, search, and export your wiki.
