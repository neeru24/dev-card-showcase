# Travel Journal & Map Tracker

A web app for logging trips, uploading photos, marking locations on a map, and writing travel stories. Features route visualization, trip stats, and export to PDF/CSV.

## Features
- Log trips with photos and stories
- Mark locations
- Route visualization
- Trip stats
- Export to CSV/PDF
- Local storage for all data

## Usage
Open index.html in your browser. Log your trips, visualize routes, and export your travel journal.
