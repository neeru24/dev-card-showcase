## ‚ú® Features

### Core Functionality
- **Recovery Logging**: Track sleep hours, stretching time, rest activities, and stress levels
- **Recovery Score Calculation**: Automated scoring based on sleep quality, stretching duration, activities, and stress management
- **Visual Score Display**: Circular progress indicator showing current recovery score
- **Historical Tracking**: View recent recovery days with detailed statistics
- **Stretching Tips**: Get expert stretching advice to improve flexibility and prevent injury
- **Local Storage**: Persistent storage of recovery logs and data

### User Experience
- **Intuitive Interface**: Clean, modern UI with recovery-focused design
- **Real-time Updates**: Scores and statistics update instantly
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Progress Visualization**: Visual representation of recovery scores and trends
- **Expert Tips**: Built-in stretching and recovery advice

### Technical Features
- **Vanilla JavaScript**: No external dependencies
- **Modular Architecture**: Well-structured, maintainable codebase
- **Performance Optimized**: Efficient DOM manipulation and data handling
- **Cross-browser Compatible**: Works on all modern browsers
- **Accessibility**: Keyboard navigation and screen reader support

## üéÆ Demo

Experience the Recovery Day Planner live: [View Demo](./index.html)

## üöÄ How to Use

1. **Log Your Recovery Day**:
   - Enter your sleep hours (recommended: 7-9 hours)
   - Input stretching time in minutes
   - Select rest activities from the dropdown
   - Set your current stress level (1-10)
   - Choose the date for the recovery day

2. **View Your Score**:
   - See your recovery score displayed in a circular indicator
   - Check average and best scores
   - Track total recovery days logged

3. **Get Tips**:
   - Click "New Tip" for stretching advice
   - Tips focus on proper stretching techniques and recovery best practices

4. **Review History**:
   - View your recent recovery days
   - See detailed information for each logged day

## üõ†Ô∏è Technical Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: Browser Local Storage
- **Styling**: Custom CSS with gradients and animations
- **Icons**: Unicode emojis for visual elements

## üèÉ Recovery Guide

### Understanding Recovery Scores
Your recovery score is calculated from four main factors:

- **Sleep (0-30 points)**: Optimal 7-9 hours = 30 points
- **Stretching (0-25 points)**: 20+ minutes = 25 points
- **Rest Activities (0-25 points)**: Each activity = 5 points (max 25)
- **Stress Level (0-20 points)**: Lower stress = higher points

### Best Practices
- Aim for 7-9 hours of quality sleep
- Stretch for at least 20 minutes daily
- Include multiple rest activities
- Monitor and manage stress levels
- Log recovery days consistently

### Stretching Tips
- Hold stretches for 20-30 seconds
- Breathe deeply and relax
- Focus on major muscle groups
- Stretch both sides equally
- Never bounce or force a stretch
