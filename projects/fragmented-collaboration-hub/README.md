# Fragmented Collaboration Hub

A simple, fast web app to unify tasks, chat, docs, and files for small teams. Designed to reduce context loss and improve collaboration.

## Features
- Task board for actionable items
- Team chat for quick discussions
- Shared docs for notes and plans
- File list (demo)
- Clean, responsive UI

## Usage
Open `index.html` in your browser. Switch between tabs to manage tasks, chat, docs, and files.

## Project Structure
- `index.html` — Main app page
- `style.css` — App styling
- `app.js` — Collaboration logic
- `README.md` — Project info and instructions
