The Complete N-Queens Guide

üìú History of the Problem

The Eight Queens puzzle was originally published in 1848 by chess composer Max Bezzel. The mathematical challenge is to place eight chess queens on an 8√ó8 chessboard so that no two queens threaten each other. Over the years, computer scientists expanded this to the N-Queens Problem, where you must place N queens on an N√óN matrix.

üìê Grid Sizes & Mathematics

Using the slider in the control panel, you can adjust the matrix size from 4x4 up to 10x10.

  • N = 4: The smallest solvable board (2 solutions).
  • N = 8: The classic chess board size (92 distinct solutions).
  • N = 10: A highly complex matrix with exactly 724 unique solutions!

Note: Grids smaller than 4x4 (like 2x2 or 3x3) have zero mathematical solutions, which is why the slider starts at 4.

  • Rows: No two queens can share the same horizontal row.
  • Columns: No two queens can share the same vertical column.
  • Diagonals: No two queens can be on the exact same diagonal path.

üéÆ How to Use This App

Visualizer Mode

Watch the computer solve the puzzle using a Backtracking Algorithm. The algorithm places a queen in the first column, then moves to the next. If it finds a safe spot, it places the next queen. If it reaches a dead end where a queen gets attacked, it "backtracks" by removing the last queen and trying a different path. You can adjust the animation speed to see the CPU "thinking" in real-time!

Puzzle Mode

Select this mode to challenge your own brain! Click on the grid cells to manually place or remove a queen.

  • Undo Move: Made a mistake? Click the orange Undo button to revert your last placement or removal.
  • Submit Answer: Once you have placed all N queens, click Submit to run the validation engine. It will highlight any attacking queens in red!
  • Show Solution: Stuck? Click this to instantly generate one of the mathematically correct answers.
