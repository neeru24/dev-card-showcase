# Portfolio & Skills Showcase

A web app for users to create a personal portfolio, highlight projects, list skills, add certifications, and display achievements. Features project gallery, skill progress bars, downloadable resume, and social links integration.

## Features
- Profile info
- Project gallery
- Skills list with progress bars
- Certifications
- Achievements
- Downloadable resume
- Social links integration
- Local storage for all data

## Usage
Open index.html in your browser. Build your portfolio, add projects, skills, certifications, and achievements.
