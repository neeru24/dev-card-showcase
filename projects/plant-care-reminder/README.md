# Plant Care Reminder

Track watering, fertilizing, and sunlight needs for houseplants.

## Features
- Add, edit, and delete plants
- Track watering, fertilizing, and sunlight tasks
- Calendar view for care tasks
- Dashboard with plant health summary
- Settings for customization
- Data persists in local storage

## How to Use
1. Open `index.html` in your browser.
2. Add your plants and set care schedules.
3. View your care calendar and dashboard.

- Export/import data
