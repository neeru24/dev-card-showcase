# Book & Media Reading Log

A web app to log books, articles, and media you consume. Features reading goals, progress bars, genre tags, cover uploads, timeline visualization, and export/share options.

## Features
- Log books, articles, movies, and more
- Set and track reading goals
- Progress bars for each entry
- Genre tags and cover uploads
- Timeline visualization
- Export to JSON/CSV
- Share reading stats
- Local storage for all data

## Usage
Open index.html in your browser. Add entries, set goals, and track your reading journey.
