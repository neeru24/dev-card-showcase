# Event Planner & RSVP Tracker

A web app to create events, send invitations, track RSVPs, manage guest lists, and visualize event timelines. Includes reminders, calendar integration, and export options.

## Features
- Event creation
- Invitation sending
- RSVP tracking
- Guest list management
- Event timeline visualization
- Reminders
- Calendar integration
- Export to CSV/JSON
- Local storage for all data

## Usage
Open index.html in your browser. Create events, manage guests, and track RSVPs.
