# Language Learning Progress Tracker

A web app to log vocabulary, grammar topics, and practice sessions. Features spaced repetition reminders, streaks, progress charts, and exportable study logs. Includes support for multiple languages.

## Features
- Log vocabulary, grammar, and practice
- Spaced repetition reminders
- Streak tracking
- Progress charts
- Export study logs (CSV/JSON)
- Multi-language support
- Local storage for all data

## Usage
Open index.html in your browser. Log your study sessions and track your language learning progress.
