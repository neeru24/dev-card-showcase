# Emotional Volatility Index

A web-based tool for tracking daily mood levels and analyzing emotional volatility patterns over time.

## Features

- **Daily Mood Logging**: Record your mood on a scale of 1-10 with optional notes
- **Volatility Analysis**: Calculate standard deviation of mood levels over 7-day periods
- **Trend Visualization**: Interactive chart showing mood trends and volatility over the last 30 days
- **Statistics Dashboard**: View average mood, current volatility, and total logged days
- **Data Persistence**: All entries stored locally in your browser
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Select a date and set your mood level using the slider
2. Add optional notes about your mood
3. Click "Log Mood" to save the entry
4. View your statistics and trends in the dashboard
5. Manage your entries by deleting old ones if needed

## Technical Details

- Built with vanilla JavaScript
- Uses Chart.js for data visualization
- Stores data locally using browser localStorage
- Responsive CSS with modern design

## Privacy

All mood data is stored locally in your browser and never transmitted to any server. Your emotional health data remains completely private.
