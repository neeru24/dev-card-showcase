# Language Learning Progress Tracker

Track vocabulary, grammar lessons, and practice sessions for language learning.

## Features
- Add vocabulary words and meanings
- Add grammar topics and notes
- Log practice sessions (date, type, minutes)
- Visualize practice time analytics (bar chart)
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Add vocabulary, grammar lessons, and log practice sessions.
3. View your progress and analytics.

## Tech Stack
- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) for analytics chart
