How It Works

This tool analyzes JavaScript code to identify:

  • Unused functions - Functions that are defined but never called
  • Unused variables - Variables that are declared but never used
  • Unused imports - Import statements that are not referenced

Note: This is a basic static analyzer and may not catch all edge cases. For production code, consider using more advanced tools like ESLint with the 'no-unused-vars' rule.
