# Dream Pattern Analyzer

A web app where users log their dreams, and the app uses simple text analysis to find recurring themes, emotions, or symbols. Over time, users can visualize patterns in their dreams and gain insights into their subconscious.

## Features
- Log and store dreams securely in your browser
- Analyze dreams for recurring words, emotions, and symbols
- Visualize dream history and top recurring patterns
- Clean, modern, and responsive UI

## Planned Advanced Features
- Tag dreams with custom labels
- Visual charts for recurring patterns
- Export/import dream logs
- Sentiment analysis and word clouds
- User authentication (optional)
- Dark mode

## How to Use
1. Open `index.html` in your browser.
2. Enter your dream description and click "Analyze Dream".
3. View analysis results and your dream history below.
