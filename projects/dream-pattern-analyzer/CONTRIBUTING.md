# Contributing to Dream Pattern Analyzer

Thank you for your interest in contributing! Here are some guidelines to help you get started:

## How to Contribute
- Fork the repository and create your branch.
- Make your changes with clear, descriptive commit messages.
- Test your changes thoroughly.
- Submit a pull request with a detailed description of your changes.

## Code Style
- Use clear, readable variable and function names.
- Write comments for complex logic.
- Keep functions small and focused.
- Use consistent indentation and formatting.

## Feature Suggestions
- Open an issue for new features or enhancements.
- Describe your idea and its benefits.

## Bug Reports
- Open an issue with steps to reproduce the bug.
- Include screenshots or logs if possible.

## Code of Conduct
- Be respectful and constructive.
- Follow the [Code of Conduct](../CODE_OF_CONDUCT.md).

---

Happy coding!
