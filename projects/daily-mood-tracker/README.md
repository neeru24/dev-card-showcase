# Daily Mood Tracker & Visualizer

An app where users log their daily mood, select emotions, and see trends visualized over time with charts or color-coded calendars.

## Features
- Log daily mood with emoji and notes
- Visualize mood trends with a colorful bar chart
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Select a date, mood, and optionally add notes.
3. Submit to add your entry.
4. View your mood history and trends in the chart below.

## Tech Stack
- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) for chart visualization
