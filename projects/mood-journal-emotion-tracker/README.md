# Mood Journal & Emotion Tracker

A web app for daily mood logging, emotion tagging, and journaling. Features mood analytics, emotion trends, calendar heatmap, and export to PDF/CSV.

## Features
- Daily mood logging
- Emotion tagging
- Journaling
- Mood analytics
- Emotion trends chart
- Calendar heatmap
- Export to CSV/PDF
- Local storage for all data

## Usage
Open index.html in your browser. Log moods, tag emotions, and track your journey.
