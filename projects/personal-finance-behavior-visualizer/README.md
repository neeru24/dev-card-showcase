# Personal Finance Behavior Visualizer

A modern, interactive web app to help you understand not just your balance, but your spending habits, triggers, and trends over time.

## Features
- Add transactions with category, trigger, and description
- Visualize spending by category, trigger, and monthly trend
- Get behavioral insights and actionable recommendations
- Clean, responsive UI

## Usage
Open `index.html` in your browser. Add your transactions, view charts, and get insights!

## Project Structure
- `index.html` — Main app page
- `style.css` — App styling
- `app.js` — Logic, charting, and insights
- `README.md` — Project info and instructions
