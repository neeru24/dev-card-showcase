# Online Learning Accountability Hub

A modern, interactive web app to help you stay consistent with online courses. Track your progress, set check-ins, and get peer/mentor support for real accountability.

## Features
- Add and track multiple courses
- Set and update progress for each course
- Log regular check-ins with reflections
- Peer and mentor support (demo)
- Clean, responsive UI

## Usage
Open `index.html` in your browser. Add your courses, track progress, log check-ins, and connect with peers/mentors!

## Project Structure
- `index.html` — Main app page
- `style.css` — App styling
- `app.js` — Logic and UI
- `README.md` — Project info and instructions
