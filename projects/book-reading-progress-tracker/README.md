# Book Reading Progress Tracker

Log books you’re reading, track progress, and set reading goals.

## Features
- Add books with title, author, total pages, and goal date
- Update current page as you read
- Visual progress bar for each book
- Track reading goals
- Responsive, modern UI
- Data saved in browser (localStorage)

## How to Use
1. Open `index.html` in your browser.
2. Add books you’re reading.
3. Update your current page as you read.
4. See your progress and goal date for each book.

## Tech Stack
- HTML, CSS, JavaScript
