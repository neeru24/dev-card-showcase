# Fitness & Workout Tracker

A comprehensive web app for tracking workouts, progress, and achievements. Includes exercise database, workout builder, logging, progress tracking, calendar, statistics, templates, badges, charts, export, and local storage.

## Features
- Exercise Database (100+ exercises)
- Workout Builder
- Workout Programs
- Exercise Logging
- Progress Tracking
- Calendar View
- Workout Templates
- Achievement Badges
- Statistics Dashboard
- Charts & Analytics
- Rest Timer
- 1RM Calculator
- Body Part Focus
- Export Data (CSV/JSON)
- Local Storage

## Usage
Open index.html in your browser. Track workouts, build routines, log progress, and export your data.
