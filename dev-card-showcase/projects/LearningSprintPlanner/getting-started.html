Step-by-Step Tutorial

Getting Started

First, explore the dashboard. You'll see your current streak, study hours, and active goals all in one place. This is your command center for all learning activities.

Step 1

Create Your First Goal

Navigate to "Study Goals" tab and click "Add New Goal". Set a realistic, measurable goal (e.g., "Complete JavaScript Basics in 2 weeks"). Goals keep you motivated and on track.

Step 2

Organize Your Topics

Go to "Topics" section and add subjects you want to study. You can categorize them by difficulty level or priority. Break down complex subjects into smaller, manageable topics.

Step 3

Start Your First Session

Click "Study Sessions" and select a topic. Choose your preferred session duration (25, 45, or 90 minutes). Use the built-in timer to maintain focus and block distractions.

Step 4

Monitor Your Progress

Visit the "Progress" tab to see visual analytics. Track topics completed, study hours logged, and maintain your daily streak. This data motivates continuous improvement.

Step 5

Test Your Knowledge

Take quizzes in "Quiz Analytics" section to assess your learning. Get instant feedback and see where you need improvement. Repeat questions to strengthen weak areas.

Troubleshooting FAQ

Data not saving to localStorage?

Issue: Your goals and study sessions disappear after refresh.

Solution: Check if localStorage is enabled in your browser. Some browsers block storage in private/incognito mode. Clear browser cache and try again. Make sure JavaScript is enabled.

Timer not working correctly?

Issue: Timer counts incorrectly or doesn't stop at zero.

Solution: Check browser console for JavaScript errors. Try closing and reopening the page. Ensure your system time is correct. Try a different browser to isolate the issue.

Progress charts not showing?

Issue: Analytics page is blank or charts don't display.

Solution: Charts require at least one study session to display. Add a topic and complete a study session first. Check if JavaScript charting library is loaded. Refresh the page.

How do I create a pull request?

Steps:
1. Fork the repository
2. Create a feature branch: git checkout -b feature/name
3. Make changes and commit: git commit -m "Add: description"
4. Push to your fork: git push origin feature/name
5. Go to GitHub and click "New Pull Request"
6. Write a clear description and submit

I messed up my commits, what do I do?

Solution: Don't worry! Before pushing:
git reset --soft HEAD~1 (undo last commit, keep changes)
git reset --hard HEAD~1 (undo last commit, discard changes)
After pushing, contact maintainers or submit a corrected PR.
