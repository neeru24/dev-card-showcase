# Habit Tracker Forest

Plant a virtual tree for each habit you want to build. Each day you complete the habit, the tree grows. Missed days cause the tree to wilt. The forest visualizes all habits and their progress.

## Features
- Add and track multiple habits
- Each habit is a tree in your forest
- Daily check-in: mark habits as done
- Trees grow with consistency, wilt if missed
- Modern, beautiful UI

## Usage
1. Open `index.html` in your browser.
2. Add habits you want to build.
3. Mark habits as done each day.
4. Watch your forest grow!
