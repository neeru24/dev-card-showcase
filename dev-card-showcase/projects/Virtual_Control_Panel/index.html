How to Use the Control Panel

  • Toggle switches to change system states. Observe how controls and indicators react.
  • Main Power must be ON to enable other controls. Turning it off will disable all other switches.
  • Safety System must be enabled before activating the Cooling System.
  • System Mode toggles between AUTO (system-controlled) and MANUAL (user-controlled) operation.
  • Use the Reset Panel button to restore all controls to their default state.
