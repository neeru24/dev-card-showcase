# Customizable Workflow Automation Platform

A platform to visually create, edit, and run automation workflows with triggers, actions, conditions, and integrations (email, calendar, social media, etc.).

## Features
- Drag-and-drop workflow builder
- Add triggers, actions, and conditions
- Integrate with popular APIs
- Attractive, modern UI
- Run and clear workflows

## Usage
1. Open `index.html` in your browser.
2. Add blocks to the workflow canvas.
3. Select integration and block type.
4. Run or clear the workflow.
