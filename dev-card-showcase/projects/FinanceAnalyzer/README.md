# AI-Powered Personal Finance Analyzer

A dashboard to securely import bank statements, categorize transactions using machine learning, visualize spending trends, and provide personalized savings/investment recommendations.

## Features
- Import CSV/Excel bank statements
- Automatic transaction categorization
- Spending trends visualization
- Personalized recommendations

## Usage
1. Open `index.html` in your browser.
2. Import your bank statement (CSV format).
3. View categorized transactions, charts, and recommendations.
