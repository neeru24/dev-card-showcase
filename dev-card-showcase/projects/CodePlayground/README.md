# Interactive Code Playground

Create, test, and share code snippets in the browser. Supports multiple languages, live preview, output console, code templates, tabs, themes, download/share/embed, and more.

## Features
- Multi-language support (JavaScript, Python, HTML/CSS, Java, C++)
- Monaco Editor with syntax highlighting
- Output console and live preview
- Code templates and multiple tabs
- Download, share, embed code
- Theme selector (Light/Dark/Monokai/Dracula)
- Keyboard shortcuts, code history, collaboration ready

## Usage
1. Open `index.html` in your browser.
2. Select language and theme.
3. Write and run code, view output and preview.
4. Download, share, or embed your code.
