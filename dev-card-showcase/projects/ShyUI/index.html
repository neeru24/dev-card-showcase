How It Works

  • Move your cursor around the page - elements will react when you get close
  • Adjust the shyness slider to change how sensitive elements are to your cursor
  • Click "Reset All Elements" to bring all elements back to their original positions
  • Try "Surprise Mode" for unexpected reactions from the UI elements
  • Some elements might hide completely, run away, or just shake nervously
