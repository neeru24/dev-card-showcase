# Goal Setting & Productivity Dashboard

A web app to set personal goals, break them into tasks, track progress, visualize productivity trends, and earn achievement badges. Includes daily/weekly planner, reminders, and export to CSV/JSON.

## Features
- Goal creation
- Task breakdown
- Progress tracking
- Productivity charts
- Achievement badges
- Daily/weekly planner
- Reminders
- Export to CSV/JSON
- Local storage for all data

## Usage
Open index.html in your browser. Set goals, add tasks, track progress, and boost productivity.
