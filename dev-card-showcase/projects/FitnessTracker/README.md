# Fitness Progress Tracker

Visualize workout routines, log exercises, track progress with charts, and set fitness goals.

## Features
- Set and view fitness goals
- Log workouts (date, exercise, reps, sets)
- View workout history
- Progress charts (total reps per day)

## Usage
1. Open `index.html` in your browser.
2. Set your fitness goals.
3. Log your workouts.
4. View progress charts.
