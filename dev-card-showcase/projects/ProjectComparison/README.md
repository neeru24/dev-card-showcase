# Project Comparison Tool

Compare up to 3 projects side-by-side on key metrics and features.

## Features
- Select up to 3 projects for comparison
- Comparison matrix with key metrics
- Visual comparison charts (radar/bar)
- Technology stack and feature checklist
- Performance and community metrics
- Learning difficulty and time investment
- Export/share/save comparison
- Winner badges

## Usage
1. Open `index.html` in your browser.
2. Select projects and compare.
3. View matrix, charts, features, and badges.
4. Export, share, or save your comparison.
