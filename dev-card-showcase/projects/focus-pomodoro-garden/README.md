# Focus Pomodoro Garden

Start pomodoro sessions and grow your garden! Each completed session plants a flower. Streaks unlock rare plants and decorations. Missed sessions cause some flowers to wilt.

## Features
- Pomodoro timer (25 minutes)
- Each completed session plants a flower
- Streaks unlock rare plants and decorations
- Missed sessions cause some flowers to wilt
- Session history list
- Modern, beautiful UI

## Usage
1. Open `index.html` in your browser.
2. Start a pomodoro session.
3. Complete sessions to grow your garden!
