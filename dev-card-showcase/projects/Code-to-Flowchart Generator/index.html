Overview

This Flowchart Generator uses Mermaid.js to render flowcharts from text-based code, similar to PlantUML.

Usage

1. Go to Generate Flowchart.

2. Write your flowchart code in the textarea.

3. Click "Render Flowchart" or see live updates.

Syntax Examples

graph TD

A[Start] --> B{Is it working?}

B -- Yes --> C[Great!]

B -- No --> D[Fix it]

D --> B

Node Types

Rectangle: Process

Diamond: Decision

Oval: Start/End

Subroutine: Subroutine
